linuxOfConfig
=============

my config on linux(such as vim /tmux etc)

####由于使用多个unbuntu系统（公司虚拟机/自己本子/落云），所以建立了这个仓库。

现在有以下几个配置文件：

1. [vim](https://github.com/hominlinx/vim)
2.  tmux

####QA

1. tmux中vim的colorscheme显示不对。解决方法：
	> 执行tmux时键入-2.可以在ubuntu的~/.bashrc中添加：
	```
		alias tmux='tmux -2'
	```
